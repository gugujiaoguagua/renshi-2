# skill 成熟度跟踪

用于跟踪每个 skill 从草稿到可导入使用的成熟过程。

## 成熟度等级

| 等级 | 名称 | 标准 |
|---|---|---|
| L0 | 临时提示词 | 只解决一次任务 |
| L1 | skill 草稿 | 有初步流程 |
| L2 | 可复用 skill | 被 2 次以上任务验证 |
| L3 | 可导入 skill | 有标准 `SKILL.md` |
| L4 | 带样例 skill | 有 examples 和质量检查 |
| L5 | 可应用化 | 高频多人使用，值得做 demo |

## 跟踪表

| skill | 当前等级 | 验证次数 | 最近使用 | 缺口 | 下一步 |
|---|---|---|---|---|---|
| 示例业务流程技能 | L1 | 0 | 待填写 | 需要真实案例验证 | 用第一个真实任务测试 |
