# 质量检查清单

任务交付前至少检查以下内容。

## 基础检查

- [ ] 是否完成用户本次请求。
- [ ] 是否读取了必要资料。
- [ ] 是否没有编造文件、字段或结论。
- [ ] 是否没有覆盖原始资料。
- [ ] 是否标明不确定项。
- [ ] 是否给出下一步建议。

## 资产包检查

- [ ] 是否需要写任务复盘。
- [ ] 是否发现新业务规则。
- [ ] 是否需要更新 memory。
- [ ] 是否需要更新规则候选。
- [ ] 是否需要创建或升级 skill。
- [ ] 是否需要进入应用化候选池。

## skill 检查

- [ ] `SKILL.md` 是否包含清晰触发描述。
- [ ] 是否有输入材料说明。
- [ ] 是否有执行流程。
- [ ] 是否有输出格式。
- [ ] 是否有质量检查。
- [ ] 是否避免塞入过多历史材料。
